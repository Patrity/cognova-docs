# Roadmap

Current version: :latest-version{repo="patrity/cognova"}

## Foundation

| Feature | Status | Details |
|---------|--------|---------|
| Database initialization | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | PostgreSQL via Drizzle ORM with auto-migration on startup |
| Authentication | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | BetterAuth with email/password, session cookies, API tokens |
| Skills system | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Python skill scripts callable from Claude Code hooks |

## Core Features

| Feature | Status | Details |
|---------|--------|---------|
| Task management | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Full CRUD with projects, priorities, tags, soft delete, and restore |
| Project management | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Full CRUD, soft delete, color coding, task/document association |
| Cron agents | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Scheduled Claude Agent SDK execution with budget caps and run history |
| Interactive chat | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | WebSocket streaming chat with Claude, conversation persistence, cost tracking |
| Multimodal chat input | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | File uploads, image paste, and drag-and-drop attachments in chat |
| CLI installer | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | `cognova init`, `update`, `start`, `stop`, `restart`, `reset`, `doctor` |
| Memory system | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Extraction from conversations, session injection, access tracking, search + skill |
| Document metadata | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Vault file sync with tags, sharing, project association, frontmatter parsing |
| Document sharing | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Public/private sharing with UUID-based access and SEO control |
| Web terminal | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Full PTY terminal via xterm.js and node-pty with session persistence |
| File system API | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Vault CRUD with path security validation |
| Dashboard | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Stat cards, upcoming tasks, recent chats/docs, usage summary |
| Full-text search | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | PostgreSQL `tsvector` indexing across tasks, documents, memories, agents, and conversations with Cmd+K search palette |
| Keyboard shortcuts | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Cmd+K search palette with navigation shortcuts |
| Secrets API | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | AES-256-GCM encrypted key-value store with settings UI |
| Settings page | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Profile, password, secrets management, notification preferences |
| Notifications | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Real-time WebSocket event bus across 9 resource types with per-action preferences |
| Message bridge | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Bidirectional messaging with Telegram and BlueBubbles (iMessage), webhook receivers, contact management |
| Hook analytics | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Claude Code hook event logging, stats, and filtering dashboard |
| Token usage tracking | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Per-source cost breakdown with daily/hourly granularity |
| System service deployment | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | launchd (macOS) and systemd (Linux) with auto-start, log management |
| Secrets skill | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | `/secret` slash command for listing, reading, writing, and deleting encrypted secrets |
| Skills management | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | Browse, enable/disable, rename, edit, create, and agent-generate skills from the web UI |
| Community skills library | :icon{name="i-lucide-check-circle" class="text-green-500"} Shipped | GitHub registry sync, one-click install/update, tag-based filtering |

## Dependency Graph

```
database-init ──┬── auth
                ├── task management
                ├── project management
                ├── full-text search
                ├── secrets API
                ├── message bridge
                └── token usage tracking

skills system ──┬── task skill
                ├── cron agents
                ├── skills management
                └── community library

secrets API ────── secrets skill

chat ───────────┬── multimodal input
                ├── conversation persistence
                └── memory extraction

auth ───────────┬── notifications
                └── message bridge
```
